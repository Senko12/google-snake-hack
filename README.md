# google-snake-hack
A hack for google snake game

# Instructions

- Go to google.com
- Search for "snake google game"
- Click "Play"
- Click some arrow to skip the "tutorial"
- Open console, paste the content of [hack.js](hack.js)
- Press enter and enjoy, the snake will follow the cursor
